Access Logger

Microserviço de telemetria web (Slim 4 + MySQL). Ambiente local Docker na porta 8088.

Comece aqui

  • Demo interativo — Copa do Mundo 2026
  • Health check (GET /health — JSON)

API (JSON)

  • POST /api/access-log — pageview
  • POST /api/access-log/update — engajamento
  • POST /api/access-log/events — eventos em lote
  • POST /api/access-log/event — evento único
  • GET /api/access-log/stats
  • GET /api/access-log/journey (troque session_id)

Se esta página abriu, o nginx está OK. O demo usa o mesmo host; a API responde nos paths acima.
